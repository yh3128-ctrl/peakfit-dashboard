## 📋 컬럼 설명

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| `산이름` | 문자열 | 기준이 된 100대 산 이름 |
| `장소유형` | 문자열 | 음식점 / 카페 |
| `장소명` | 문자열 | 상호명 |
| `카테고리` | 문자열 | 카카오맵 세부 카테고리 (예: `한식 > 육류,고기요리`) |
| `주소` | 문자열 | 도로명 주소 |
| `전화번호` | 문자열 | 매장 전화번호 |
| `산으로부터거리(m)` | 정수 | 산 좌표로부터의 직선 거리 (단위: 미터) |
| `도보시간(분)` | 정수 | 거리 ÷ 75m/분으로 계산한 도보 예상 시간 |
| `장소위도` | 실수 | 장소 위도 좌표 |
| `장소경도` | 실수 | 장소 경도 좌표 |
| `카카오지도URL` | 문자열 | 카카오맵 상세 페이지 링크 |
| `블로그언급수` | 정수 | 네이버 블로그 검색 언급 수 (인기도 지표) |
| `맛집점수` | 실수 | **100점 만점 종합 점수** (핵심 컬럼) |

## ⭐ 맛집점수 산정 기준

맛집점수는 아래 4가지 요소를 합산한 **100점 만점** 점수입니다.

| 항목 | 배점 | 세부 기준 |
|------|------|-----------|
| 블로그 언급 수 | 최대 40점 | 언급 많을수록 높음 (log 스케일 적용) |
| 거리 접근성 | 최대 30점 | 500m 이내 30점 / 1,500m 이내 20점 / 3,000m 이내 5점 |
| 전화번호 유무 | 15점 | 전화번호 있으면 15점 (정식 영업 여부) |
| 카테고리 세분화 | 15점 | 세부 카테고리 depth 3 이상이면 15점 (전문점 여부) |

### 점수 구간별 해석

| 점수 구간 | 장소 수 | 해석 |
|-----------|---------|------|
| 80 ~ 100점 | 194건 | ⭐⭐⭐ 강력 추천 맛집 |
| 60 ~ 80점 | 832건 | ⭐⭐ 검증된 맛집 |
| 40 ~ 60점 | 1,160건 | ⭐ 무난한 선택 |
| 20 ~ 40점 | 503건 | 정보 보통 |
| 0 ~ 20점 | 51건 | 정보 부족 |

| 장소유형 | 건수 |
|----------|------|
| 음식점 | 594건 |
| 카페 | 265건 |
| **합계** | **859건** |

## 💡 활용 방법

### 1. 하산 후 바로 갈 맛집 찾기
```
산으로부터거리(m) ≤ 1500  AND  맛집점수 ≥ 60
```

### 2. 특정 산 주변 맛집 보기
```
산이름 = "설악산"  →  맛집점수 내림차순 정렬
```

### 3. 카페만 보기
```
장소유형 = "카페"  AND  맛집점수 ≥ 40
```

### 4. 지도에서 바로 확인
- `카카오지도URL` 컬럼의 링크를 클릭하면 카카오맵에서 바로 확인 가능

---

## ⚠️ 데이터 한계

- 산 **정상 좌표** 기준으로 수집되어 실제 탐방로 입구와 거리 차이가 있을 수 있음
- 블로그 언급 수는 인기도의 **간접 지표**이며 음식 품질을 직접 반영하지 않음
- 카카오맵 API 특성상 한 산당 최대 **45건**까지만 수집됨 (카테고리별 15건 × 3페이지)
- 폐업했으나 지도에 남아있는 장소가 일부 포함될 수 있음

## 🛠 수집 환경

| 항목 | 내용 |
|------|------|
| 수집 API | 카카오맵 Local API, 네이버 블로그 검색 API |
| 검색 반경 | 각 산 기준 3km |
| 카테고리 코드 | `FD6` (음식점), `CE7` (카페) |
| 언급수 쿼리 | `{장소명} {시/군} 맛집` 키워드로 네이버 블로그 검색 |
